# Features

This directory contains Gherkin `.feature` files that define the business requirements and user scenarios for the Event Venue Booking App.

These files serve as:

- **Requirements Documentation**: A clear description of how the system should behave.
- **Living Documentation**: A source of truth for stakeholders and developers.
- **Test Specifications**: The basis for automated acceptance tests.

## Files

- `list_venue.feature`: Scenarios related to venue owners listing and managing their venues.
